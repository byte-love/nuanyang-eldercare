# 如何补充 GIF / 更多截图（可选）

README 已包含登录页、首页看板静态图。若希望简历效果更好，建议用 **ScreenToGif** 或 **OBS** 录制以下页面（10～15 秒即可）：

| 文件名 | 建议录制内容 |
|--------|----------------|
| `docs/images/dify-chat.gif` | 暖阳智询：发送问题 → 流式回复 |
| `docs/images/health-assessment.gif` | 健康评估：上传报告 → 生成结果 |
| `docs/images/check-in.gif` | 入住办理流程 |

保存后，在 `README.md` 对应章节取消注释或添加：

```markdown
<p align="center">
  <img src="docs/images/dify-chat.gif" alt="智能咨询" width="800" />
</p>
```

Git 提交：

```bash
git add docs/images/
git commit -m "docs: add demo gifs"
git push
```
